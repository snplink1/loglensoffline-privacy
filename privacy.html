LogLensOffline

Privacy Policy

Effective date: May 17, 2026

Privacy Summary

LogLensOffline is a local, offline log analysis tool. It does not require a login, create an account, collect telemetry, show ads, or upload log content.

Data Collection

LogLensOffline does not collect, transmit, sell, share, or store personal information.

The app does not use subscriptions, cloud sync, analytics, advertising networks, or third-party tracking.

Log Inputs

Users may paste log text or choose local files for analysis. The app reads those inputs locally on the user's device. It does not upload log files, pasted text, or analysis results to a server.

Before entries are retained in memory for analysis, the app redacts common sensitive values such as email addresses, phone numbers, Social Security numbers, credit card-like numbers, authorization values, JWT-like tokens, common secret assignments, and local user-profile path segments.

Local Storage

The current application does not persist pasted log text, imported log content, analysis results, user identity, recent files, or usage history. Analysis data exists only for the active app process and disappears when the app closes.

Network Use

The runtime analysis workflow does not make network requests.
